# **App Name**: LinkTree EDU

## Core Features:

- Link Display: Display a series of clickable image links.
- Link Configuration: The admin can configure the links to display: title, URL, image and custom border color.
- Copyright Footer: Display a custom copyright message in the footer.
- Floating Element Removal: AI tool to automatically remove unwanted floating elements.

## Style Guidelines:

- Primary color: Dark slate gray (#334155) to provide a solid base that will give the design a very reliable, serious feel.
- Background color: Very dark slate gray (#1e293b) background with a blurred custom image and dark overlay (85% opacity).
- Accent color: Steel blue (#94a3b8) for subtle highlights.
- Font: 'Poppins', a sans-serif font for a geometric, precise feel. This font will be used for the main 'LINKS' header as well as for the copyright in the footer.
- Centralized layout with a maximum width of 400px for optimal mobile and desktop viewing.
- Image links as clickable items with scale hover effects.
- Pulse animation on the first link to draw initial attention.
